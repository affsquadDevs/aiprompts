# Prompt data sources

This catalog is assembled from openly-licensed prompt collections plus
industry "playbook" packs generated by `scripts/build-data.mjs`.

| Source | License | Use |
|---|---|---|
| [f/awesome-chatgpt-prompts](https://github.com/f/awesome-chatgpt-prompts) (`prompts.csv`) | CC0 1.0 (public domain) | Core "act as" prompt corpus |
| [abilzerian/LLM-Prompt-Library](https://github.com/abilzerian/LLM-Prompt-Library) | MIT | Category prompts (finance, legal, medical, programming…) |
| [mustvlad/ChatGPT-System-Prompts](https://github.com/mustvlad/ChatGPT-System-Prompts) | MIT | System / roleplay prompts |

Generated playbook packs are original content produced by this repo's build script.
